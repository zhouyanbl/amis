### Password

密码输入框。

-   `type` 请设置成 `password`
-   `addOn` 输入框附加组件，比如附带一个提示文字，或者附带一个提交按钮。
-   `addOn.type` 请选择 `text` 、`button` 或者 `submit`。
-   `addOn.label` 文字说明
-   `addOn.xxx` 其他参数请参考按钮配置部分。
-   `clearable` 在有值的时候是否显示一个删除图标在右侧。
-   `resetValue` 默认为 `""`, 删除后设置此配置项给定的值。
-   更多配置请参考 [FormItem](./FormItem.md)
